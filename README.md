# 📌 Recipy - AI-Powered Recipe Recommendation System

Recipy is a **Flask-based web application** that suggests recipes based on the ingredients you have. It also provides **allergen-free alternatives** using machine learning techniques. Built as part of a **summer internship after the first year**, this project showcases a **practical implementation of AI in food technology**.

## 🚀 Features
- 🍽️ **Recipe Suggestions** – Get personalized recipes based on available ingredients.
- 🛑 **Allergen-Free Substitutions** – Automatically replaces common allergens like peanuts, milk, eggs, etc.
- 🧠 **AI-Powered Matching** – Uses **TF-IDF vectorization** and **cosine similarity** to find the best recipe matches.
- 📊 **Data-Driven Approach** – Processes a dataset of curated recipes and ingredient lists.
- 🌎 **Web-Based Interface** – Built with **Flask, HTML/CSS**, and deployed on **Render**.

## 🔧 Tech Stack
- **Backend:** Flask, Python, Pandas, Scikit-Learn
- **Frontend:** HTML, CSS
- **Deployment:** Render
- **Machine Learning:** TF-IDF, Cosine Similarity
